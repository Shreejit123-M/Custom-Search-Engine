# Custom Search Engine

A Simple Flask-based custom search engine that takes a search engine that takes a search query and fetches relevant results.


## Features

- Web interface for search
- Custom search logic using Python
- Simple UI built with HTML templates

## Technologies Used

- Python
- Flask
- BeautifulSoup
- Requests
- HTML/CSS (Jinja templates)
